Number Guessing Game in C

📌 Overview

This is a simple console-based Number Guessing Game developed in C. The program generates a random number between 1 and 100, and the player attempts to guess it. After each guess, the program provides hints indicating whether the guess is too high or too low. The game also counts the number of attempts and allows the player to play multiple rounds.

✨ Features

* Random number generation between 1 and 100
* User-friendly hints (Too High / Too Low)
* Input validation for invalid entries
* Tracks the number of attempts
* Option to play multiple rounds

🛠️ Technologies Used

* C Programming Language
* Standard C Libraries (`stdio.h`, `stdlib.h`, `time.h`)

▶️ How to Run

1. Compile the program:

   ```
   gcc number_guessing_game.c -o game
   ```
2. Run the executable:

   ```
   ./game
   ```

   (On Windows: `game.exe`)
